Devel generate extras
=====================

This module is inspired by the Devel issue [Let's generate kick-ass demo content!](https://drupal.org/node/1748302), and aims to make it possible to use [Devel](https://drupal.org/project/devel)'s `devel_generate` to generate more realistic demo content.

Usage
-----

This module is not yet fully functional but aims to package the code in [Let's generate kick-ass demo content!](https://drupal.org/node/1748302).
